## 🍪 Cookie Clicker Automation Bot

A concise and focused overview of the project’s purpose and behavior, written to be informative without exposing implementation details.

---

### 🎯 Purpose  
This project automates gameplay in **Cookie Clicker**, simulating continuous interaction and strategic decision‑making to maximize cookie production within a defined session.

---

### ⚙️ Key Features  
- **Automated interaction** with the main game element to generate resources consistently.  
- **Smart item selection** based on cost efficiency and progression impact.  
- **Priority‑driven resource management** to maintain steady growth.  
- **Initial environment setup** including automatic language configuration.  
- **Performance tracking** through end‑of‑session production metrics.

---

### 🧠 Automation Strategy  
The bot follows a streamlined logic designed for efficiency:

- **Continuous clicking** to maintain constant resource generation.  
- **Automatic upgrade acquisition** whenever beneficial improvements become available.  
- **Item purchasing logic** prioritizing high‑value options.  
- **Timed decision cycles** to balance production and spending.  
- **Automatic session termination** with a final performance summary.

---

### 📊 Final Output  
At the end of the session, the bot retrieves the **Cookies Per Second (CPS)** value, providing a clear indicator of overall efficiency.
